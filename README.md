# Tic-Tac-Toe Game

Tic-Tac-Toe game for 2 human players to play locally

(written in Python3)

It looks like this:

```
       Tic-Tac-Toe
    Game: 2 out of 5
    Score: X:1 - O:0
    ~~~~~~~~~~~~~~~~~
    O|X|        7|8|9
    -+-+-       -+-+-
     |O|O       4|5|6
    -+-+-       -+-+-
    X| |X       1|2|3

O's Turn [1-9]:
```

Enjoy!!!
